# 🌾 AgriSense AI

## AI-Powered Smart Farming Assistant

AgriSense AI is an intelligent farming assistant that helps farmers detect crop diseases using Artificial Intelligence. It provides disease identification, treatment recommendations, fertilizer suggestions, weather-based alerts, disease history tracking, voice assistance, and multilingual support.

---

## Features

- 🌱 Crop Disease Detection
- 🦠 Disease Identification
- 💊 Treatment Recommendation
- 🌿 Fertilizer Recommendation
- 🌦 Weather-based Alerts
- 📚 Disease History
- 🎙 Voice Assistant
- 🌍 Multi-language Support
